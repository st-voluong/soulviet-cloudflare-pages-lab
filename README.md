# SoulVietLab

SoulVietLab is a modern static travel landing page for the idea "Discover hidden places with local stories". The website introduces SoulViet for travelers who want authentic Vietnam experiences through hidden destinations, cultural activities, and local stories.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Local mock data
- Static hosting ready

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is generated in:

```bash
dist
```

## Preview Production Build

```bash
npm run preview
```

## Deployment

Deployment target: Cloudflare Pages

- Build command: `npm run build`
- Output directory: `dist`
- Planned custom domain: `soulviet.luongvanvo.id.vn`
